Responsive Design using CSS Media Queries

What is Responsive Design?

Responsive Web Design (RWD) is a technique used to make websites adapt to different screen sizes such as desktops, tablets, and mobile phones.

Desktop

Large screens display cards in multiple columns.

Tablet

Medium screens display fewer columns.

Mobile

Small screens display cards in a single column.
