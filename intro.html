Introduction to HTML, CSS & JavaScript

The three core technologies used for Front-End Web Development.

HTML (HyperText Markup Language)

HTML is the standard markup language used to create the structure of web pages.

Common HTML Elements

  • <html>
  • <head>
  • <body>
  • <h1> to <h6>
  • <p>
  • <a>
  • <img>
  • <table>
  • <form>
  • <div>
  • <span>

CSS (Cascading Style Sheets)

CSS is used to style HTML elements and make websites attractive.

Common CSS Properties

  • color
  • background-color
  • font-size
  • font-family
  • margin
  • padding
  • border
  • width & height
  • display
  • flexbox
  • grid
  • position
  • box-shadow
  • border-radius

JavaScript

JavaScript is a scripting language used to make web pages interactive.

JavaScript Concepts

  • Variables
  • Data Types
  • Operators
  • Functions
  • Conditions
  • Loops
  • Arrays
  • Objects
  • Events
  • DOM Manipulation
